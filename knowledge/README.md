# Knowledge Layer

This directory is a public-safe knowledge scaffold for the framework.

It intentionally contains:

- mentor-brain sublibrary skeletons
- domain profile examples

It intentionally does **not** contain:

- private mentor raw materials
- confidential project brains
- generated runtime artifacts

For real usage, private mentor/project knowledge should stay outside the public repo and be built into artifacts locally.

本目录是公共安全的知识层脚手架。

- 导师脑子库骨架
- 领域 profile 示例

它不会放：

- 私有导师原始材料
- 私有项目脑
- 运行时生成产物
