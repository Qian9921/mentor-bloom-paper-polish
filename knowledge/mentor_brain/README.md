# Mentor Brain Sublibraries

This directory sketches how a mature mentor brain can be decomposed into maintainable sublibraries instead of a single monolithic profile.

Recommended sublibraries:

- `lexicon.yaml`
- `forbidden_phrases.yaml`
- `sentence_patterns.yaml`
- `transition_patterns.yaml`
- `paragraph_contracts.yaml`
- `section_contracts.yaml`
- `revision_operators.yaml`
- `figure_narration_style.yaml`

These files are intentionally lightweight public-safe scaffolds. Real content should be built from private mentor materials through the mentor-bloom workflow.

这里展示一个更成熟的导师脑拆分方式：

- 词汇库
- 禁忌表达
- 句法模式
- 转折模式
- 段落契约
- section 契约
- 改稿动作库
- 图表叙述风格

真实内容应通过私有导师材料本地构建，而不是直接把私有导师数据提交到公共仓库。
