# Mentor Exemplar Agent Consumption Protocol v0.1

目标：把导师的高质量正向样例提取成可检索的 exemplar index，让 agent 不只是知道“不要怎么写”，还知道“应该怎样写”。

---

## 1. 分层原则

- script：从 mentor corpus 中筛出高价值 exemplar
- agent：在写作/改稿时检索并消费这些 exemplar

script 不应直接把 exemplar 强行改写进用户稿件。

---

## 2. exemplar 的用途

exemplar 主要用于：

- abstract opening
- contribution sentence
- result takeaway
- figure/table introduction
- limitation/future-work pairing
- conclusion landing

---

## 3. agent 应做的事

agent 看到 exemplar 后，应回答：

1. 这个 exemplar 对当前段落有什么可借鉴的 rhetorical move？
2. 当前段落应该借它的结构、句法，还是词汇？
3. 哪些地方不能机械照搬？
