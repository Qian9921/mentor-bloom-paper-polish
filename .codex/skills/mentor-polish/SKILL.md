---
name: mentor-polish
description: Agent-first mentor-style revision workflow using revision agenda, reference mirror, and pass-based workspaces.
---

<Purpose>
Mentor Polish turns an xray report and revision agenda into a structured revision workflow. Scripts package pass briefs and section tasks; Codex/agents perform the actual rewriting.
</Purpose>

<Canonical_Rule>
Scripts package. Agents rewrite.
</Canonical_Rule>

<Typical_Flow>
1. Build or load a mentor-approach report (score + gaps)
2. Prepare revision agenda
3. Build revision packet
4. Build reference mirror packet + brief
5. Build mentor-polish brief + pass workspace
6. Let Codex/agent revise by pass (macro, paragraph, sentence, terminology, verify)
</Typical_Flow>
