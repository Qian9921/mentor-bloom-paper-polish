---
name: paper-xray
description: Agent-first diagnosis workflow for mature research drafts using mentor profile, term checks, and lightweight truth guards.
---

<Purpose>
Paper Xray packages a mature draft into an agent-consumable diagnosis workflow. Scripts prepare the draft map, term findings, truth-check-lite findings, and packet; Codex/agents author the actual xray report.
</Purpose>

<Canonical_Rule>
Scripts prepare. Agents diagnose.
</Canonical_Rule>

<Typical_Flow>
1. Build draft map
2. Build project fact sheet
3. Build paper xray packet + brief
4. Have Codex/agent write `xray_report.json`
</Typical_Flow>
